# Marketplace Starter

Use this skill when the user wants to understand, adapt, validate, or publish this starter plugin as a GitHub-hosted marketplace plugin.

## Checklist

1. Confirm the public plugin name.
2. Update `plugins/<plugin-name>/.codex-plugin/plugin.json`.
3. Add real skills, MCP servers, app manifests, scripts, or assets.
4. Update `.agents/plugins/marketplace.json`.
5. Validate the plugin manifest.
6. Publish the repository to GitHub.
7. Ask users to register the marketplace and install the plugin.
