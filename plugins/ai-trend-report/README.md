# AI Trend Report Plugin

This Codex plugin helps create sourced PDF reports about AI trends from the latest seven days of web news.

## Contents

- `.codex-plugin/plugin.json`: Codex plugin manifest
- `skills/ai-trend-report/SKILL.md`: Research and report workflow
- `scripts/render_report.py`: Helper that converts Markdown to styled HTML and optionally prints PDF with Chrome or Edge

The skill guides Codex to search with varied Korean and English queries, compare sources, extract trends, and produce a PDF-ready report.
